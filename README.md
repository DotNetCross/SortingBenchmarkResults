# SortingBenchmarkResults
Benchmark results for Sorting.

## Schema for Directory Name for Results
```
<YYMMDD>_<COMMITHASH>_<CPUMODEL>_[NAME]
```
where `<>` are required and `[]` are optional. Example:
```
180205_c168e6ff_i5-3475S_nietras
```
within this all files from benchmark run should be copied e.g.:
